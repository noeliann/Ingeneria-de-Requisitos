# CodeCare - Acceptance Tests 

Este repositorio contiene los archivos `.feature` correspondientes a las pruebas de aceptación del proyecto **CodeCare**, desarrollados como parte del Trabajo Final del curso **Ingeniería de Requisitos**.  


## Contenido 

Cada archivo `.feature` representa una historia de usuario e incluye:

* Definición de la funcionalidad (*Feature*). 
* Escenarios de aceptación (*Scenario* y *Scenario Outline*). 
* Datos de prueba mediante **Data Tables** y **Examples**, cuando corresponde. 
* Criterios de aceptación expresados utilizando la sintaxis estándar de Gherkin.
